# React Card Component with Props

## Overview

In this assignment, I created a reusable card component using React. The main goal was to understand how props work and how to display dynamic data from an array of objects.

## Features

* Created a reusable **Card component**
* Used **props** to pass data like title, description, and image
* Displayed multiple cards using an array and `map()` function
* Designed cards with a **gradient border**
* Maintained a clean layout using grid
* Each card displays:

  * Image
  * Title
  * Description

## Technologies Used

* React JS
* JavaScript
* HTML
* CSS (inline styling)

## How to Run

1. Open the project folder in VS Code
2. Install dependencies:
   npm install
3. Start the project:
   npm run dev
4. Open the local host link shown in the terminal
